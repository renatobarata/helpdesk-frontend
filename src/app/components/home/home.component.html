Help Desk Dicas de um Dev

Olá! Seja bem vindo ao sistema Help Desk desenvolvido por mim no curso Desenvolvedor Full Stack.

Esse é um sistema para gerenciamento de chamados simulando um simples Help Desk desenvolvido em Angular com Spring Boot 2.

O sistema está rodando na nuvem, sendo o front na Vercel e o back no Heroku.

O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autenticação com tokens JWT.

O código fonte do projeto pode ser acessado clicando no link Github no menu.

Fique a vontade para testar o sistema e deixar sugestão.
